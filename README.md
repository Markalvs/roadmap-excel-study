# Excel Completo: Do Básico ao Avançado

Repositório educacional dedicado ao estudo de Microsoft Excel, abrangendo desde os fundamentos das planilhas eletrônicas até recursos avançados de análise de dados, automação e otimização.

---

# Sobre o Projeto

Este repositório reúne conteúdos teóricos, exercícios práticos e projetos aplicados para o desenvolvimento de competências em Excel, com foco em produtividade, análise de dados, construção de relatórios e automação de processos.

O material foi organizado em módulos progressivos, permitindo a construção de conhecimento de maneira estruturada.

## Competências Desenvolvidas

- Excel Básico
- Excel Intermediário
- Excel Avançado
- Análise de Dados
- Business Intelligence
- Construção de Dashboards
- Automação de Processos
- Gestão Operacional
- Modelagem Financeira
- Pesquisa Operacional
- Tomada de Decisão Baseada em Dados
